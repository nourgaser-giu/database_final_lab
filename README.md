# HrLabApp

This is a small demo project built with .NET for a database course final lab. Students use this web application to connect to their Microsoft SQL Server database and demonstrate basic CRUD operations.

## Features

- .NET web application
- Connects to Microsoft SQL Server
- Simple database interactions

## Purpose

Designed for educational use in a database course to help students practice building web apps with database connectivity.

For more details, see the `Docs/` directory.
